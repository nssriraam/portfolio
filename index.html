About Myself

I'm a Computer Science undergraduate at United Institute of Technology, Coimbatore — focused on blue team cybersecurity with a clear path toward SOC operations and cloud security monitoring.

My work spans Linux and Windows log analysis, AWS CloudTrail monitoring, Splunk and Wazuh SIEM deployment, and MITRE ATT&CK mapping across realistic home lab scenarios. I document everything like a real analyst would.

Most recently I built an AI-powered CloudTrail anomaly detector — a Python tool using a locally hosted LLM that automates SOC-style triage across a 20-event simulated attack chain covering brute force, privilege escalation, defense evasion, and data exfiltration.

Goal: Enter the industry as a Junior SOC Analyst, grow into cloud security long-term.
